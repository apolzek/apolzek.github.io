## commit message

- [ADD] <article> <date> (new article)
- [FIX] <article> <date> (fix article)
- [UPD] <article> <date> (append article)
- [DEL] <article> <date> (delete article)

Ex:

```
git commit -m "[ADD] article-test.md 2024/05/22 about testing"
```
