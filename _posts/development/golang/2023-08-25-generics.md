## Generics

Resumindo Generics.. normalmente criamos uma variável de um tipo específico e em algum momento colocamos um valor dentro dela respeitando esse tipo. Quando utilizamos generics fazemos o inverso, onde recebemos um valor sem ter um tipo pré-definido e em algum momento podemos especifica-lo(normalmente no momento de uso).

**Generics** tem tudo haver com **Constraints e Interfaces**. Antes da versão 1.18(alguma coisa) para fazer o tipo Any era utilizado interface vazia, representado por *interface{}*.

```go
func Print(a ...interface{}) (n int, err error)
```

### Before generics existed

Para descobrir o tipo, era utilizado o **switch**:

```go
package main

import "fmt"

func something(x interface{}) {
	switch idk := x.(type) {
	case int:
		fmt.Println("eh int:", idk)
	case string:
		fmt.Println("eh string:", idk)
	default:
		fmt.Println("eh invalido", idk)
	}
}

func main() {
	something("brazil")
	something(1337)
	something(12.123)
}

```

### After generics was implemented

```go
package main

import "fmt"

type UserTest1 struct {
	name string
	age  int16
}

type UserTest2 struct {
	documentNumber string
	address        string
}

func userTest[T UserTest1 | UserTest2](user T) {
	fmt.Println(user)
	fmt.Println(user)
}

func identity[T any](arg T) T {
	return arg
}

func anyType(arg any) any {
	return arg
}

func main() {

	u1 := UserTest1{
		name: "Mario",
		age:  29,
	}
	userTest(u1)

	// num := 10
	// userTest(num)

	out1 := identity[string]("test")
	fmt.Println(out1)
	out2 := identity[int16](1234)
	fmt.Println(out2)

	out3 := anyType(1234)
	fmt.Println(out3)
	out4 := anyType("test")
	fmt.Println(out4)
}
```

E para fazer comparações entre variáveis do tipo Any ?? Abaixo está um jeito feio de se fazer
```go
func main() {
	var value1 interface{} = 42
	var value2 interface{} = "42"

	if reflect.TypeOf(value1).Kind() == reflect.Int && reflect.TypeOf(value2).Kind() == reflect.String {
		convertedValue1 := value1.(int)
		convertedValue2 := value2.(string)

		if convertedValue1 == 42 && convertedValue2 == "42" {
			fmt.Println("As variáveis são iguais.")
		} else {
			fmt.Println("As variáveis são diferentes.")
		}
	} else {
		fmt.Println("Tipos incompatíveis para comparação.")
	}
}
```

Abaixo está uma implementação mais elegante. Ele utiliza o **comparable**.
```go
package main

import "fmt"

func anyTest[T comparable](arg1 T, arg2 T) {
	fmt.Println(arg1 == arg2)
}
func main() {
	anyTest(1, 2)
}
```

Comparação com < e >. Também possível comparar tipo primitivo de tipo definido(usando **~**)
```go
package main

import "fmt"

func test[T NumberTests](arg1 T, arg2 T) {
	fmt.Println(arg1 >= arg2)
}

func testInt[T ~int64](arg1 T, arg2 T) {
	fmt.Println(arg1 >= arg2)
}

type MyNum int64

type NumberTests interface {
	int64 | float64 | float32
}

type NumberIntTests interface {
	int64
}

func main() {
	test(3.2, 2.0)

	var iNum MyNum = 44
	testInt(iNum, 33)
}

```

É possível utilizar na declaração de um tipo também. Onde não é mais necessário definir o tipo no momento da criação. Ex:
```go 
package main

import "fmt"

type User[T any, B any] struct {
	name T
	age  B
}

func main() {

	userTest := User[string, int64]{
		name: "joao",
		age:  20,
	}

	fmt.Println(userTest)
}
```
