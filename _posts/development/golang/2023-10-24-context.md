## Context

In Go (or Golang), the context is a powerful tool that allows you to control the execution of concurrent and parallel operations. It is commonly used to handle deadlines, cancel operations, and values associated with a particular goroutine (a lightweight thread of execution in Go). Context is especially useful in situations where multiple goroutines are being executed and you need to coordinate or manage their behavior. Contexts basically use channels under the hood, in which the processes that use them need to listen for signals they receive from this context.

### Where to use ?

- Context de tracing
- Context do request
- Context em query SQL

### 3 context types

**Deadlines**: tempo maximo para o codigo terminar de ser executado(definir o ponto final)
1) WithDeadline: passa o horario final para terminar(horario maximo)
2) WithTimeout: daqui a quanto tempo vai terminar

**Cancellation Signals**: todos que estiverem escutando o contexto sao cancelados
1) WithCancel

**Request-scoped values**
1) WithValue: definir e mapear valores(em memoria)

### 1. Controlando Prazos (Timeouts):
```go
package main

import (
    "context"
    "fmt"
    "time"
)

func main() {
    ctx, cancel := context.WithTimeout(context.Background(), time.Second*2)
    defer cancel()

    select {
    case <-time.After(time.Second * 3):
        fmt.Println("Operação demorou muito!")
    case <-ctx.Done():
        fmt.Println("Operação cancelada devido ao prazo expirado.")
    }
}

```

### 2. Cancelamento de Operações:\
```go
package main

import (
    "context"
    "fmt"
    "time"
)

func worker(ctx context.Context) {
    for {
        select {
        case <-ctx.Done():
            fmt.Println("Goroutine cancelada.")
            return
        default:
            fmt.Println("Trabalhando...")
            time.Sleep(time.Second)
        }
    }
}

func main() {
    ctx, cancel := context.WithCancel(context.Background())

    go worker(ctx)

    // Aguarde por algum tempo e então cancele a operação
    time.Sleep(time.Second * 3)
    cancel()

    // Aguarde um pouco para ver a saída
    time.Sleep(time.Second)
}
```

### 3. Comunicação de Valores entre Goroutines:

```go
package main

import (
    "context"
    "fmt"
    "sync"
)

type key string

func worker(ctx context.Context, wg *sync.WaitGroup) {
    defer wg.Done()

    // Recupera o valor associado à chave do contexto
    userID := ctx.Value(key("userID")).(string)
    fmt.Println("Goroutine recebeu UserID:", userID)
}

func main() {
    // Crie um contexto com um valor associado à chave "userID"
    ctx := context.WithValue(context.Background(), key("userID"), "12345")

    var wg sync.WaitGroup

    // Inicie duas goroutines que compartilham o mesmo contexto
    for i := 0; i < 2; i++ {
        wg.Add(1)
        go worker(ctx, &wg)
    }

    // Aguarde até que todas as goroutines terminem
    wg.Wait()
}
```

### Context - WithCancel
```go
package main

import (
	"context"
	"fmt"
	"time"
)

func main() {
	ctx, cancel := context.WithCancel(
		context.Background(),
	)
	go printUntilCancel(ctx)

	time.Sleep(3 * time.Second)

	cancel()

	time.Sleep(10 * time.Second)
}

func printUntilCancel(ctx context.Context) {
	count := 0
	for {
		select {
		case <-ctx.Done():
			fmt.Println("Cancel signal received, exiting")
			return
		default:
			fmt.Printf("I'm here\n")
			time.Sleep(1 * time.Second)
			fmt.Printf("Printing until cancel, number = %d \n", count)
			count += 1
		}
	}
}
```

import (

### Context - WithTimeout
```go
package main

import (
	"context"
	"fmt"
	"sync"
	"time"
)

func main() {
	ctx, cancel := context.WithTimeout(
		context.Background(),
		5*time.Second,
	)

	// defer cancel()
	// printUntilCancel(ctx)
	var wg sync.WaitGroup
	wg.Add(1)

	go printUntilCancel(ctx, &wg)
	time.Sleep(3 * time.Second)
	cancel()
	wg.Wait()
}

func printUntilCancel(ctx context.Context, wg *sync.WaitGroup) {
	defer wg.Done()
	count := 0
	for {
		select {
		case <-ctx.Done():
			fmt.Println("Cancel signal received, exiting")
			return
		default:
			time.Sleep(1 * time.Second)
			fmt.Printf("Printing until cancel, number = %d \n", count)
			count += 1
		}
	}
}

```

### Context - WithValue
```go
package main

import (
	"context"
	"fmt"
)

func main() {
	ctx := context.WithValue(
		context.Background(),
		"testKey",
		"testValue",
	)

	printUntilCancel(ctx)
}

func printUntilCancel(ctx context.Context) {
	ctx2 := context.WithValue(ctx, "testKey", "palmeiras")
	fmt.Println(ctx2.Value("testKey"))
	fmt.Println(ctx.Value("testKey"))
}

```
