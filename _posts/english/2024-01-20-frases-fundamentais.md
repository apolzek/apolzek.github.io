## Frases fundamentais

Cumprimentos
```
Hey!
Hi there!
What's up?
Howdy!
Hey, how's it going?
Yo!
Hiya!
Hey, what's going on?
Sup?
Hello, mate!
```

Desculpas
```
I'm sorry
That's ok # Ok não foi nada

Excuse me
```

Pedir algo
```
Can I have a pen, please ?
May I have a cup of water ?
Could I have a pencil? ?

Sure - Claro
Here - Ta aqui
Here, it is
```

Elogiar
```
I like your hair
```

Esperar algo
```
Just a sec
Just a second
Just a minuto
```

Agradecer
```
You're welcome
No problem
No prob
```

Pedir para falar mais devagar 
```
Could you please speak a bit slower ? 
Could you please spell that ?
Could you please type that ?
```

Quando não entendeu algo
```
I'm sorry. I didn't get that - Desculpe eu não entendi
Sorry, I don't get it
I don't know that word

Ok, gotcha ! / I got you - Entendi
Oh really !
That makes sense
```

Eu acho / Eu não acho
```
I think so
I don't think so
```
