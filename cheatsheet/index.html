  • Send Text using ICMP networkicmp

    Encode and send text payload via ping, receive and decode with tshark

    # Send
payload=$(echo -n "Hello" | xxd -p)
echo $payload
len=$(($(echo $payload | wc -c | tr -s " ") + 2))
echo $len
ping -s$len -c4 -p$payload 127.0.0.1
# Receive
tshark -l -i lo0 -Y "icmp" -T fields -e data | xxd -r
  • Chrome Bookmarks (filter by folder) chromejqlinux

    Extract URLs from a specific Chrome bookmark folder using jq

    cat ~/.config/google-chrome/Default/Bookmarks \
  | jq -r '.roots.bookmark_bar.children[]
      | select(.name == "tree")
      | .children[]
      | select(.name == "blogs")
      | .children[]
      | .url'

  • Encrypt / Decrypt File with Password opensslsecuritylinux

    Use OpenSSL AES-256-CBC with PBKDF2 to encrypt and decrypt files

    # Encrypt
echo "Este é um arquivo de teste" > original.txt
openssl enc -aes-256-cbc -salt -pbkdf2 -in original.txt -out secreto.enc

# Decrypt
openssl enc -aes-256-cbc -d -salt -pbkdf2 -in secreto.enc

# Decrypt from remote URL
curl -s https://raw.githubusercontent.com/apolzek/apolzek.github.io/main/secreto.enc \
  | openssl enc -aes-256-cbc -d -salt -pbkdf2
